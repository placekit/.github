Welcome to our GitHub organisation! PlaceKit is a geocoding API providing fast and accurate address autocomplete, store locator, and two-way geocoding for your app.

This is our public developers space, where we share our API clients and reference that we've carefully documented. You'll find integration examples in in all client repositories. We also try to maintain a global changelog here and keep it digestable, so you get a big picture on the efforts we make into improving the service.
